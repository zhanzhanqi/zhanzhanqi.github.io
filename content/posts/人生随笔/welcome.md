# 欢迎来到我的博客

这是一个支持 **Markdown** 语法的个人博客系统。

## 功能特点

- 支持完整的 Markdown 语法
- 静态生成，加载速度快
- 简洁优雅的设计风格

## 开始使用

点击右上角的「写文章」按钮，开始创作你的第一篇文章吧！

### 代码示例

```javascript
function hello() {
  console.log('Hello, World!');
}
```
